## SaaS Funnel Analytics

End-to-end analytics project simulating a SaaS product.

### Features
- Synthetic user, event, subscription, payment data
- Funnel analysis (Signup → Activation → Paid)
- Weekly cohort retention (0–12 weeks)
- Churn analysis
- DuckDB + SQL + Python

### Tech Stack
- Python
- DuckDB
- SQL
- Pandas
- Matplotlib
